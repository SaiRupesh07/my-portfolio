## 🚀 Run Locally

```bash
# Clone the repo
git clone https://github.com/SaiRupesh07/my-portfolio.git

# Navigate into folder
cd my-portfolio

# Install dependencies
npm install

# Start dev server
npm run dev
```

Open **http://localhost:5173** in your browser 🎉
